# Build and Conquer

**Build and Conquer** is a mod for *Gladius* that enhances city expansion mechanics.  
With this mod, the **main city structure** gains the ability to automatically acquire surrounding tiles, making territorial growth faster and more strategic.

## Features
- Cities can claim adjacent tiles directly through their primary structure.
- Accelerated territorial expansion compared to the base game.
- Encourages aggressive building and conquest strategies.

## Installation
1. Download or clone this repository.  
2. Place the mod files into your *Gladius* mods folder.  
3. Enable **Build and Conquer** in the game’s mod manager.
